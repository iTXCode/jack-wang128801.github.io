Linux 基本操作

在Linux终端和 shell中复制与粘贴不能用 Ctrl +c y与 Ctrl+v

要用Ctrl+Insert 和Shift +Insert 来完成复制与粘贴

ls / 进入当前机器的根目录(/: 根目录)

ll 的结果的第一个字母为 - 表示不同文件，为 d 表示为目录

cd 总结

cd .. : 返回上级目录

cd /home/litao/linux/ : 绝对路径

cd ../day02/ : 相对路径

cd ~：进入用户家目录(cd 啥都不加也是同样的效果)
